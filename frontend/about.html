Project Title

Fog-Enabled Intelligent Safety Gear for Industrial Workers

This project focuses on enhancing industrial worker safety using a fog-enabled IoT architecture. The system leverages edge computing for immediate hazard detection and fog computing for intelligent analysis, digital twin management, and real-time visualization in low-connectivity environments such as mines.
